# Laravel-Vue Trello App
### System Requirements
1) PHP version: 8.1 
2) Node version: 12.13.0
3) NPM version: 6.12.0
4) Laravel version: 8.1
### Installation Steps
1) `composer install`
2) `npm install` 
3) `php artisan key:generate`
4) `php artisan migrate`
5) `php artisan db:seed`
6) `npm run watch`
7) `php artisan serve`
